## About

**GitReplay** is a free browser app for exploring public GitHub repositories. Paste a repo link, browse the file tree, and replay files as they type out on screen. Web projects also show a **live preview** as HTML, CSS, and JavaScript are built.

Nothing to install. Open the link and start.

---

## Quick start

1. Go to [delexoo.github.io/GitReplay/welcome.html](https://delexoo.github.io/GitReplay/welcome.html)
2. Paste a repo URL (for example `github.com/vitejs/vite`)
3. Click **Load**
4. Single-click a file to read it. Double-click to replay it live.

---

## What you can do

- Browse any public repo file tree
- Replay files with typing animation
- Preview websites, markdown, images, and media
- Replay all files or race HTML, JS, and CSS files
- Pause, change speed, and scrub progress during replay
